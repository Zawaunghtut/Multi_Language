၅ ဘာသာ အလွယ်ဆုံး Translator (Multi-Language Translator)
တစ်ကြိမ်တည်းနဲ့ ဘာသာစကား ၅ မျိုးကို အလွယ်တကူ ဘာသာပြန်ဆိုနိုင်ရန် ဖန်တီးထားသော Tool ဖြစ်ပါသည်။ ဤ Project သည် HTML, CSS နှင့် JavaScript တိုကို အသုံးပြု၍ ရိုးရှင်းပြီး အသုံးပြုရလွယ်ကူသော User Interface ဖြင့် တည်ဆောက်ထားပါသည်။
အဓိက လုပ်ဆောင်ချက်များ (Features)
 * Multi-Language Support: စာသားတစ်ခုကို ရိုက်ထည့်ရုံဖြင့် ဘာသာစကား ၅ မျိုးသို တစ်ပြိုင်နက် ဘာသာပြန်ပေးခြင်း။
 * User-Friendly Interface: မြန်မာဘာသာဖြင့် ရှင်းလင်းစွာ ဖော်ပြထားသော Menu များနှင့် ခလုတ်များ။
 * Responsive Design: မိုဘိုင်းဖုန်းနှင့် ကွန်ပျူတာ Screen နှစ်မျိုးလုံးတွင် ကြည့်ကောင်းအောင် ပြုလုပ်ထားခြင်း။
 * Fast Translation: "ဘာသာပြန်ရန်" ခလုတ်ကို နှိပ်လိုက်ရုံဖြင့် လျင်မြန်စွာ အဖြေထွက်ပေါ်ခြင်း။
အသုံးပြုထားသော နည်းပညာများ (Tech Stack)
 * HTML5: Website ၏ တည်ဆောက်ပုံအတွက်။
 * CSS3: ဒီဇိုင်းပိုင်းနှင့် Gradient Background များအတွက်။
 * JavaScript: ဘာသာပြန်စနစ်နှင့် အပြန်အလှန်လုပ်ဆောင်မှုများအတွက်။
အသုံးပြုနည်း
၁။ "ဘာသာပြန်လိုသည့် စာသားကို ရိုက်ထည့်ပါ" ဟူသော နေရာတွင် မိမိဘာသာပြန်ချင်သော စာကို ရေးပါ။
၂။ "ဘာသာပြန်ရန်" ခလုတ်ကို နှိပ်ပါ။
၃။ သတ်မှတ်ထားသော ဘာသာစကား ၅ မျိုးဖြင့် အဖြေများ ထွက်ပေါ်လာပါလိမ့်မည်။
